2.0 Build Instructions
----------------------

2.1 Compilers
The following compilers and development environments were used to
create this software:
Borland C++ Version 5.01

2.2 Libraries
TGrid - This library is distributed with QConBride source. Please note
that this is shareware. If you are going to develop QConBridge source code
you will need to pay the auther his registration fee ($25).

2.3 Build Instructions
Build the various components in the order specified below
WSDOTLIB\CODE
WSDOTLIB\ESL
WSDOTLIB\MATERIAL
WSDOTLIB\SHAPE
WSDOTLIB\ROARK
WSDOTLIB\FEM2D
OWLTOOLS
BRIDGE
CBA (QConBridge)
